# How to run

 1. Install Python3
     * If you have chocolatey just run 'cinst python3'
     * Otherwise download from [here](https://www.python.org/downloads/release/python-374/)

2.  Install libtcod
    * run 'py -m pip install tcod'

3. Clone the repo
    * Run ' git clone https://github.com/nebiont/rogue'

4. Run 'py .\engine.py'
